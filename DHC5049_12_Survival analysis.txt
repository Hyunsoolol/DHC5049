  데이터 관측기간의 길이에 따라 구분

  Cross-sectional data (횡단 자료)
  : 특정 시점 또는 짧은 기간동안 한번 수집한 자료
  예. 국민건강영양조사 자료

       특정 시점에서의 상태나 연관성 분석                          South Korea." Scientific Reports 11.1 (2021): 24470.

    예. 2025년 3월 현재 고혈압 여부에 따른 비만율 분석

  Longitudinal data (종단 자료)
  : 동일한 대상을 여러 기간에 걸쳐 측정하거나 여러 시점에 걸쳐 반복측정한 자료

      시간에 따른 변화나 인과관계 분석
      예. 2020년부터 2025년까지의 고혈압 여부에 따른 비만 위험도 분석
      예. 한국의 COVID-19 pandemic 동안의 일별 발생건수 패턴 변화 분석

Framingham Heart Study라는 코호트 연구에서 과거 당뇨병 진단 여부 (DIABETES)와 심뇌혈관질환 (CVD) 발생 위험
간의 연관성을 확인하고자 한다.

이때 Period 1에 해당하는 대상군 (PERIOD=1)에 한하여,
전체 연구기간에서 당뇨병 발생여부에 따른 심뇌혈관질환 발생의 생존곡선이 다른지 여부를 통계적으로 검정하라.


 가설       𝐻0 : 𝑆1 𝑡 = 𝑆2 𝑡 for all 𝑡 (모든 시점에서 두 군의 생존확률은 같다)    유의수준   0.05
          𝐻1 : 𝑆1 𝑡 ≠ 𝑆2 𝑡 for some 𝑡



 검정법      양측 이표본 로그순위 검정 (two-sided two-sample log-rank test)
 양측 (two-sided): 차이 검정
 이표본 (two-sample): 2개의 독립 집단 간의 비교
 Log-rank 검정 : 생존자료에 대한 검정통계량의 확률분포

library(dplyr)

# CVD 발생 비율 및 time-to-event 확인
with(tmp, table(DIABETES, CVD))
with(tmp, tapply(TIMECVD, DIABETES, summary))

with(tmp, table(DIABETES, CVD))
with(tmp, tapply(TIMECVD, DIABETES, summary))

  회귀계수 𝛽𝑘 의 의미

    𝐻0 ∶ 𝛽𝑘 = 0     𝑯𝑹𝒌 = 𝟏   ℎ 𝑡 | 𝑋𝑘 = 1 = ℎ 𝑡 | 𝑋𝑘 = 0
     No difference in hazard
     No association between 𝑌 & 𝑋𝑘

    𝐻0 ∶ 𝛽𝑘 ≠ 0 𝑯𝑹𝒌 ≠ 𝟏 ℎ 𝑡 | 𝑋𝑘 = 1 ≠ ℎ 𝑡 | 𝑋𝑘 = 0
     Significant difference in hazard
     Significant association between 𝑌 & 𝑋𝑘

  회귀 계수 𝛽𝑘 의 부호
      𝛽𝑘 > 0      𝐻𝑅 𝑋𝑘 > 1   ℎ 𝑡 | 𝑋𝑘 = 1 > ℎ 𝑡 | 𝑋𝑘 = 0   : 𝑋𝑘 = 1 일때 odds ↑ (risky)
      𝛽𝑘 < 0      𝐻𝑅 𝑋𝑘 < 1   ℎ 𝑡 | 𝑋𝑘 = 1 < ℎ 𝑡 | 𝑋𝑘 = 0   : 𝑋𝑘 = 1 일때 odds ↓ (protective)

  회귀 계수 𝛽𝑘 의 절대값
     회귀 계수의 절대값이 클수록 해당 설명변수의 영향력/예측력이 크다는 의미
     HR값이 1과 차이가 많이 날수록 해당 설명변수의 영향력/예측력이 크다는 의미
